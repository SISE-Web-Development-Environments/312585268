Who is Avatar Aang?

Avatar Aang is the last AirBender,
Long ago, the four nations lived together in harmony.
Then, everything changed when the Fire Nation attacked.
Only the Avatar, master of all four elements, could stop them, but when the world needed him most, he vanished.
A hundred years passed and the new Avatar, an airbender named Aang was found, and although his airbending skills are great, he has a lot to learn before he's ready to save anyone.

    Question 1

    The 3 key diffrences between Web 1.0, Web 2.0 and Web 3.0 are:

    • User Interaction - which means in Web1.0 is not interactive with the user, it is only a bridge between the websites and the user for example look at craiglist.com.
      while in Web2.0 and Web3.0 are fully interactive with the user.

    • Search - In Web1.0 the user must use the full URL of the site which he wants to access to, while in Web2.0 and Web3.0 the user can use only the base URL.

    • Readbility - Web1.0 is Mostly Read-Only, and Web2.0 is Wildly Read-Write while Web3.0 is Portable and Personal.

    Question 2

    Searching for "medicine side effects"

    • In Web1.0 the user will write a specific url of the site he knows is talking about the medicine side effects

    • In Web2.0 the user will search the query above in the search engine (such as google.com), which will return every link that have tags or words related to the query.

    • In Web3.0 the user will search the query above in the search engine, which will return every link or part of a page that contains words or tag from the query.
